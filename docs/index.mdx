## What is ASO++?

ASO++ is an MCP server that brings App Store Optimization data directly into the AI tools you already use — Cursor, Claude Code, VS Code, and more. Instead of switching between dashboards, you can research keywords, audit your app, and track rankings without leaving your editor.

## Available MCP tools

Once connected, your AI assistant can call these tools on your behalf:

| Tool                      | What it does                                             |
| ------------------------- | -------------------------------------------------------- |
| `search_keywords`         | Find ASO keywords for a given query and country          |
| `get_keyword_popularity`  | Get popularity score and trend for a keyword             |
| `get_app_ranking`         | Check where your app ranks for a keyword                 |
| `get_app_details`         | Fetch app metadata: title, description, ratings          |
| `get_competitor_keywords` | Discover keywords a competitor ranks for                 |
| `get_app_reviews`         | Retrieve recent reviews for any app                      |
| `track_keyword`           | Start tracking a keyword for your app                    |
| `get_tracking_history`    | View ranking history for tracked keywords                |
| `aso_audit`               | Run a full ASO audit and get prioritized recommendations |
